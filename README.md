# Apziva AI Residency — Abdulrahman Wahdan

This repository holds my work as an **AI Resident at [Apziva](https://www.apziva.com/)**, an applied machine learning program where residents take real-world problems end to end: framing the question, cleaning the data, building and tuning models, and translating the results into recommendations a business could act on.

I'm a Business Administration graduate (Finance major, Data Science minor) from the American University of Sharjah, working toward analyst and data science roles. Each folder here is a self-contained project with its own README, notebook, and findings.

## What's in each project

Every project folder follows the same structure so it's easy to navigate:

- A **README** explaining the problem, the approach, the results, and the reasoning behind the key decisions.
- The **notebook** with the full analysis, from data checks through modeling and conclusions.
- The **dataset**, where it can be shared.

## Tools

Python, pandas, NumPy, scikit-learn, XGBoost, statsmodels, SciPy, matplotlib, seaborn.

## A note on how I work

I try to make the *reasoning* as visible as the result. On the first project that meant deciding what to do about duplicate survey rows and then measuring the consequence of that decision rather than hiding it, choosing a significance test suited to the data, and separating the conclusions that are stable from the numbers that are fragile. The models matter, but so does knowing what to trust.
